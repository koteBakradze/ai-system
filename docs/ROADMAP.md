# AI_SYSTEM Roadmap

Last updated: 2026-05-12

## Phase 0 - Context Preservation

Status: In progress.

Keep repo memory, ChatGPT context, generated project summaries, and next-action files accurate so future sessions do not lose project state.

## Phase 1 - Real Internet Research Gateway

Status: Implemented; closure verification remains.

Completed capabilities:
- explicit mock/offline research;
- explicit DDGS-backed real research;
- saved source metadata reports;
- compact research context packs;
- tests for provider behavior, report writing, fallback behavior, and context extraction.

Closure checks:
- run unit tests;
- run mock research smoke check;
- run real research smoke check when network is available;
- keep docs and `.codex/` memory synchronized.

## Phase 2 - Multi-Model Review

Status: Planned.

Improve local draft, local critique, optional free OpenRouter review, and final merge workflows while keeping usage tracked and budget-safe.

## Phase 3 - Income Research Workflow

Status: Next major product phase.

Use the research gateway to discover realistic additional-income ideas, score them by time, risk, difficulty, and income potential, then create manual weekly reports.

## Phase 4 - GitHub Project Builder

Status: Planned.

Turn selected ideas into local project scaffolds, READMEs, issue plans, and MVP implementation steps. Public repo creation and pushes require explicit approval.

## Phase 5 - Safe Browser Automation

Status: Later.

Only consider visible, approval-based browser automation after the research and income workflow foundations are reliable. Do not access secrets, cookies, private accounts, or perform sensitive actions without explicit approval.
